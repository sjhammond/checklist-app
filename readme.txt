currentPhase is an integer that corresponds to each deployment phase in the checklist. 
Its value starts at 0 because it corresponds to an array position.

0 = Preparing and Installing
1 = Confiugring and Organizing
2 = Defining and Monitoring
3 = Extending and Maintaining


currentProduct is an integer that corresponds to the deployment's XProtect product tier. 

1 = Essential+
2 = Express+
3 = Professional+
4 = Expert
5 = Corporate
